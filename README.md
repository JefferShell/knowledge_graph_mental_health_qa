## 项目简介

基于知识图谱的心里健康问答系统是一个基于Django框架和Neo4j图数据库的Web应用系统。该系统可用于直观地展示和查询心理健康相关的数据。用户可以利用图谱进行数据的探索和分析，并通过自然语言提问获得与图谱数据相关的答案。

## 主要功能

* **图谱展示**：可展示知识图谱，包括不同类型的节点和它们之间的关系。
* **自然语言问答**：支持通过自然语言提问，并获得与图谱数据相关的答案。

## 技术架构

**后端:**
* Django
* Neo4j

**前端:**
* HTML
* CSS
* JavaScript

## 使用说明

1. 打开项目网站。
2. 利用图谱展示功能探索数据。
3. 输入自然语言问题以获取答案。
